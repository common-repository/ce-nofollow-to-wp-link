**Translations**

* English
* German

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

**Installation via Wordpress**

1. Go to the menu 'Plugins' -> 'Install' and search for 'CE Nofollow to WP-Link'
1. Click 'install'

**Manual Installation**

1. Upload folder `ce-nofollow-on-link` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
